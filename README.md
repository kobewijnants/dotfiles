# My DOTFILES

## Requirements

### GIT

```bash
sudo apt install git

```

### Stow

```bash
sudo apt install stow
```

## Installation

Go to $HOME and clone the repo

```bash
git clone https://github.com/kobewijnants/dotfiles.git
cd dotfiles
```

Then use GNU Stow to create symlinks

```bash
stow .
```
